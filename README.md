# Hero Legends Thai — Playable Demo

A browser prototype of **Hero Legends Thai** — a Phra Aphai Mani (พระอภัยมณี) hero-collection gacha RPG (Seven Knights-style), landscape, Thai UI.

- **`index.html`** — Title → Hub → Heroes (24) → Summon → Stage Select → Team Builder → Battle, plus Arena/Guild/Shop/Mail/Events/Modes/Hero-Upgrade/Equipment.
- **`battle.html`** — live battle viewer running the real deterministic engine (`battle-simulator.ts`, bundled) on the team you pick.

Art: cartoon character set generated via ChatGPT Images. Engine + gamedata mirror the NestJS backend. Landscape-only.
